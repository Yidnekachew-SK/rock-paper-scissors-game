# Rock Paper Scissors Game
This project is based on the Odin Project foundations lesson, Rock Paper Scissors project. 

The game is the implementation of the popular game rock paper scissors. It is played by a user and computer, where the user enters their choice by clicking one of the buttons
which has option for rock, paper or scissors and the computer chooses randomly. The winner is decided when the user or computer score 5 points by beating the other based on the rules of the actual game.

### Guide to play the game.
1. Open this [this live demo page.](https://yidnekachew-sk.github.io/rock-paper-scissors-game/)
2. Choose your option for the game from the three buttons
3. Continue playing until the winner is displayed
4. Refresh the page to start new game
